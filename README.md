# FISIOCEN ALBACETE S.L. (Landing estática)

Web de una sola página (HTML/CSS/JS) preparada para subir a GitHub Pages.

## Estructura

- `index.html`
- `styles.css`
- `script.js`

## Publicar en GitHub Pages

1. Crea un repositorio en GitHub.
2. Sube estos 3 archivos a la raíz del repo.
3. Ve a **Settings > Pages**.
4. En **Build and deployment**, selecciona:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (root)
5. Guarda y espera el despliegue.

## Notas de contenido

- No se incluyen horario, email ni web porque no están confirmados.
